¿Quieres aprender a crear tu propio espacio digital?

En este curso aprenderás a desarrollar tu propio sitio web desde cero, usando herramientas y lenguajes de programación básicos.

Duración: 3 horas totales

¿Qué aprenderás?

✔️ Estructura básica de una página web con HTML.

✔️ Estilización de páginas con CSS y diseño responsive.

✔️ Introducción a JavaScript para interacción en el sitio.

Este curso será impartido por la ingeniera Laura Rodríguez.
